## Functional Requirements

### Member features
- Check-in and check-out
- Book group classes
- Log workouts manually
- Sync health data from fitness bands/watches
- View personalized exercise plans
- Join community discussions & leaderboards
- Manage personal profile
- View billing & subscriptions
### Employee features (Web app)
- Manage members
- View attendance
- Create and manage group classes
- Moderate community content
- Access branch-level analytics
### System features
- Support multiple gym branches
- Handle payments and subscriptions

## Assumptions
